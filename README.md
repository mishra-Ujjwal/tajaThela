# 🛒 Taja Thela – Local Marketplace Platform

A multi-role marketplace platform connecting vendors, customers, and delivery partners with real-time, location-based product discovery.

---

## 📌 Overview

Taja Thela is a **three-sided marketplace system** designed to digitize local street vendors and enable customers to discover nearby products efficiently.

The platform focuses on **real-time availability, location-based filtering, and role-based dashboards** for seamless operations.

---

## 🎯 Problem Statement

Local vendors often lack:

* Digital visibility
* Efficient order management systems
* Access to nearby customers

Customers face:

* Difficulty in finding nearby vendors
* Lack of real-time product availability

---

## 🚀 Solution

Taja Thela provides a **centralized marketplace** where:

* Vendors can list and manage products
* Customers can discover vendors based on location
* Delivery partners can manage order fulfillment

---

## ⚡ Key Features

* 📍 **Geolocation-Based Filtering**
  Filter vendors within a **5–10 km radius** for faster and relevant results

* 👥 **Multi-Role System**
  Separate dashboards for:

  * Vendors
  * Customers
  * Delivery Partners

* 🧾 **Product & Order Management**
  Vendors can manage inventory and orders efficiently

* 📊 **Role-Based Dashboards**
  Tailored UI for each user type

* ⚡ **Optimized Data Retrieval**
  Reduced query time by **~22%** using efficient data handling
